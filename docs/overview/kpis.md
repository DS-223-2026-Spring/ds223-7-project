# KPIs

## Primary Metrics

| KPI | Value | Target | Trend |
|-----|-------|--------|-------|
| Conversion rate | 5.4% | ≥ 5% | +2.1% vs baseline |
| Avg time to convert | 6.2 days | ≤ 7 days | −3.8 days vs control |
| 30-day Pro retention | 83% | ≥ 80% | ✓ on target |

## Segment-Level Metrics

Each of the four user segments tracks a funnel:

| Segment | Colour | Focus |
|---------|--------|-------|
| Power | Green `#00a86b` | Upsell — high engagement, ready to convert |
| Growing | Blue `#3b82f6` | Nurture — rising usage, needs a push |
| Casual | Amber `#f59e0b` | Re-engage — low frequency, price-sensitive |
| Dormant | Grey `#9ca3af` | Win-back — inactive users, churn risk |

## Funnel Steps (per segment)

1. **Impression** — campaign message delivered
2. **Open** — user opens / views the message
3. **Click** — user clicks the CTA
4. **Convert** — user upgrades to Pro

## Global Parameters

Tunable parameters exposed in the Campaign Editor:

- `pro_price_amd` — Pro plan price in AMD (default: 2,900)
- `dormant_discount` — discount % offered to dormant users
- `template_count` — number of active message templates per campaign
