# Problem

## Context

**Armat** is a SaaS platform operating a freemium model with 4,420+ free-tier users. Converting free users to paid (Pro) subscribers is the core business challenge.

## Problem Statement

Armat lacks a systematic, data-driven mechanism to identify which free users are most likely to convert and to deliver the right message through the right channel at the right moment. Without segmentation and targeted campaigns, conversion rates remain low and revenue opportunities are missed.

## Root Causes

- **No user segmentation** — all free users are treated identically regardless of engagement behaviour
- **No campaign tooling** — no internal interface for PMs to create, launch, or A/B test conversion campaigns
- **No conversion analytics** — no real-time visibility into what drives free-to-paid upgrades

## Measured Impact

| Metric | Current Baseline |
|--------|-----------------|
| Free users | 4,420+ |
| Conversion rate | 5.4% (+2.1% vs prior baseline) |
| Avg time to convert | 6.2 days (−3.8 days vs control) |
| 30-day Pro retention | 83% (target: 80% ✓) |

## Solution: Pulse

**Pulse** is a free-to-paid conversion platform that:

1. Segments users into four behavioural groups — **Power**, **Growing**, **Casual**, **Dormant**
2. Provides a Campaign Editor so PMs can write, target, and launch conversion messages per segment
3. Runs A/B tests on message variants and measures lift
4. Surfaces all KPIs in a real-time analytics dashboard
